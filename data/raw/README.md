# Raw data

This folder stores small raw input files that can be shared with the repository.

It does not contain the full raw CHELSA-TraCE21k climate archive and does not contain the restricted archaeobotanical workbook.

## Contents

| Subfolder | Description |
|---|---|
| `regions/` | Africa and southern Africa boundary files used for regional summaries, raster extraction, and plotting. |

## Not included here

- Raw CHELSA-TraCE21k climate rasters. These must be downloaded separately from the CHELSA data portal.
- Restricted archaeobotanical site-level data. These are stored locally under `data/restricted/` only and are not uploaded to GitHub.
