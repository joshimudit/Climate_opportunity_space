# Outputs

This folder contains final figures and tables produced by the analysis scripts.

Outputs are separated into:

| Folder | Contents |
|---|---|
| `figures/` | Main manuscript figures and selected supplementary figures. |
| `tables/` | Final result tables and supplementary tables. |

## Notes

- Large intermediate outputs, exploratory plots, logs, and restricted site-level tables are not included unless they are needed for reproducibility and are safe to share.
