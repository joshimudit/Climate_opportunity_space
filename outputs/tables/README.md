# Tables

This folder contains final result tables and selected supplementary tables.

## Files

| File | Description |
|---|---|
| `parameter_sensitivity.xlsx` | Summary table from the EcoCrop parameter-sensitivity analysis. |
| `gam_effects.xlsx` | Corrected GAM effect-size table based on `prop_0.8`. |
| `gam_kcheck.xlsx` | GAM k-check diagnostics. |
| `gam_smooths.xlsx` | GAM smooth summary table. |
| `threshold_sensitivity.xlsx` | Threshold-dependent climate sensitivity table. |
| `arch_summary.csv` | Aggregate archaeobotanical evaluation summary. |
| `arch_effects.csv` | Archaeobotanical model-data effect-size summary. |
| `archeobotanical_evaluation.xlsx` | Combined archaeobotanical evaluation workbook. |
